- **Confidential:** Financial details, account numbers

3. Regular backups of all documents
4. Version control for important changes

- **Primary:** Git repository (this location)
- **Secondary:** Cloud backup (Google Drive/Dropbox/similar)
- **Tertiary:** Physical copies of critical documents
